markdown# Vite & Gourmand 🍽️

Application web pour le traiteur bordelais Vite & Gourmand. Développée dans le cadre du TP Développeur Web et Web Mobile (Studi).

## Description

Vite & Gourmand est une entreprise de traiteur basée à Bordeaux depuis 25 ans. Cette application permet de consulter les menus, passer des commandes et gérer les prestations.

## Fonctionnalités

- Page d'accueil avec présentation de l'entreprise et avis clients
- Liste des menus avec filtres dynamiques (prix, thème, régime, nombre de personnes)
- Détail de chaque menu avec allergènes et conditions
- Formulaire de commande avec calcul automatique du prix et des frais de livraison
- Inscription et connexion utilisateur
- Page de contact
- Mentions légales et CGV

## Installation en local

1. Cloner le repository :
git clone https://github.com/nassimbelgacem21-gif/vite-gourmand.git

2. Ouvrir le dossier :
cd vite-gourmand

3. Ouvrir `index.html` dans votre navigateur — aucune installation supplémentaire requise pour le front-end.
